## 2. 导数
**`导数: 函数图上过某一点切线的斜率`**

```
时间和距离关系: s(t) = t^3
当 t (秒), 经过 dt (秒), 到 t + dt (秒) 后, 算出来当前速率?

- t(秒): 距离 s(t)
- t + dt (秒): 距离 s(t + dt)
- ds = s(t + dt) - s(t)
- dv = f'(x) = ds / dt
```
